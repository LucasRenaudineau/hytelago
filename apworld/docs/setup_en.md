# Hytale Multiworld Setup Guide

## Required Software

- Hytale (game client)
- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases) (latest release)

## Installation

1. Download and install Archipelago.
2. Place `Hytale.apworld` into your `lib/worlds` folder inside the Archipelago installation directory.
3. Generate a multiworld using the provided YAML template.
4. Connect your Hytale client to the Archipelago server using the host, port, slot name, and password from your room page.

## Gameplay Notes

- Locations are unlocked by completing in-game achievements (kills, crafting, collection milestones).
- Items received from the multiworld will appear in your inventory.
- The goal is to defeat the **Frost Dragon**.
